見積時間編集

  1. 見積時間クリア
  2. 一桁削除
  3. 各数字入力（0〜9）
  4. 時間-分区切り入力
    5. ※分のみで入力しても時間:分の形式に変換されます。
    　例：90と入力→1:30
  5. →保存して戻る

プロジェクト編集

  1. →プロジェクトを設定して戻る：行選択
  2. 編集モード切替
    • プロジェクト追加：New Project行選択
    • プロジェクト名編集：行選択
    • プロジェクト削除：削除ボタンタップ

タグ編集

  1. タグ設定：Evernoteタグ一覧から選択、または直接入力
  2. →保存して戻る

スケジュール時刻編集

  1. スケジュール時刻クリア
  2. 一桁削除
  3. 各数字入力
  4. 時間-分区切り入力
    5. ※2桁入力すると、自動入力される
  5. →保存して戻る

サイクル数設定

  1. サイクル数クリア
  2. 一桁削除
  3. 各数字入力（0〜9）
  4. →保存して戻る

メモ編集

  1. Mail送信：メモ選択→Send

  3. Facebook送信：メモ選択→Share
  4. Twitter送信：メモ選択→Tweet

実行カレンダー

  1. 今日に移動

  4. →タスク詳細：テーブル行選択

リピート分析

  • Count（総実行回数）
  • Sum（実行時間合計）
  • Avg（平均実行時間）
  • Days（実行日数）
  • Duration（実行期間）※初回実行日〜最終実行日
  • Interval（平均実行間隔）
  • 各Sectionでの実行回数分布
  • 各曜日での実行回数分布
